
## Fleet System

This Project contains 3 sub-projects or modules, located under modules folder.

 1. Common Module
 2. Driver Module
 3. Trip Module

To run **Driver Module** navigate to root directory and execute: 

    sbt driver/run

To run **Trip Module** navigate to root directory and execute:

    sbt trip/run

To run all test cases

    sbt test
    
To run all test cases for specific module
    
    sbt <MODULE_NAME>/test
